# 🎬 Hybrid Movie Recommendation System

An end-to-end Movie Recommendation System built using **Content-Based Filtering**, **Collaborative Filtering (SVD)**, and a **Hybrid Recommendation Engine**. The project is deployed as an interactive Streamlit application where users can receive personalized movie recommendations.

---

## 📌 Project Overview

Recommendation systems are widely used in platforms like Netflix, Amazon, Spotify, and YouTube to personalize user experiences.

This project implements two popular recommendation techniques:

- **Content-Based Filtering**
- **Collaborative Filtering (Matrix Factorization using SVD)**

These two approaches are then combined into a **Hybrid Recommendation System** to generate more relevant recommendations.

---

## 🚀 Features

- 🎯 Content-Based Recommendation using TF-IDF and Cosine Similarity
- ⭐ Collaborative Filtering using Singular Value Decomposition (SVD)
- 🔥 Hybrid Recommendation Engine

# 🧠 Recommendation Pipeline

## 1. Content-Based Filtering

Movie metadata is converted into numerical vectors using **TF-IDF**.

```
Genres
        ↓
TF-IDF Vectorizer
        ↓
Feature Vectors
        ↓
Cosine Similarity
        ↓
Top Similar Movies
```

---

## 2. Collaborative Filtering

User preferences are learned using **Matrix Factorization (SVD)**.

```
User Ratings
        ↓
SVD
        ↓
Latent User Factors
+
Latent Movie Factors
        ↓
Predicted Ratings
```

---

## 3. Hybrid Recommendation

The recommendations generated by both approaches are combined to provide the final recommendation list.

```
          User

             │

 ┌───────────┴───────────┐

 │                       │

Content-Based      Collaborative

 │                       │

 └───────────┬───────────┘

             │

      Hybrid Recommendation

             │

     Final Movie Suggestions
```

# ▶️ Installation

Clone the repository

```bash
git clone <repository-url>
```

Navigate to the project

```bash
cd Movie-Recommendation-System
```

Create virtual environment

```bash
python -m venv venv
```

Activate environment

Windows

```bash
venv\Scripts\activate
```

Mac/Linux

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Train Models

Run

```bash
python train.py
```

This generates

- tfidf.pkl
- cosine_similarity.pkl
- svd_model.pkl

inside the **models/** folder.

---

# ▶️ Run Application

```bash
streamlit run app.py
```
